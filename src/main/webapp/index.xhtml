Github : https://github.com/jittagornp/income

ข้อตกลงการนำไปใช้

ผมอนุญาตให้สามารถนำโปรเจ็คนี้ไปต่อยอด ทำโปรเจ็คตนเองเพื่อการค้าขายได้ครับ
แต่ผมไม่อนุญาตให้นำโปรเจ็คนี้ ไปขายเป็น ตัวอย่าง/สื่อการสอน เพราะว่าผมทำแจกฟรี
หากใครที่ต้องการดัดแปลงเพื่อเป็น ตัวอย่าง/สื่อการสอน เพิ่มเติม ผมขอให้แนบเครดิตให้ผมด้วย

หวังว่าคนที่นำไปใช้ จะทำตามข้อตกลงน่ะครับ
